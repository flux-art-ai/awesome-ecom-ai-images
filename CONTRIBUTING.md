# 贡献指南 Contributing

欢迎补充条目,PR 需满足:

1. 工具/平台条目:官网可访问;功能描述能在其官网页面验证;不含无出处的运营数据(用户数/出图量等一律不写,除非给官方来源链接)。
2. 价格相关一律写"以官网当前为准",不写死数字。
3. 教程/工作流条目:步骤可复现,提示词可直接复制。
4. 法规条目:只链官方发布页。
5. 每个 PR 一个主题;中文条目附一句话英文说明更佳。
